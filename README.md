## 📖 About

**Shadcn Dashboard** is a simple, open-source **admin dashboard template** built on top of **Next.js**, **shadcn/ui**, **Radix UI**, and **Tailwind CSS**. It's designed as a clean starting point for developers who want to build a **React admin panel**, **SaaS dashboard**, **analytics dashboard**, or **internal tool UI** without starting from scratch.

If you're searching for a **shadcn/ui dashboard example**, a **Next.js admin template**, or a **free open-source dashboard boilerplate**, this project gives you a production-ready layout you can customize in minutes.

## ✨ Features

- ⚡ **Next.js App Router** — fast, modern React framework with server-side rendering
- 🎨 **shadcn/ui components** — accessible, unstyled, copy-paste UI primitives built on Radix UI
- 💨 **Tailwind CSS** — utility-first styling, fully customizable theme
- 🟦 **TypeScript** — type-safe codebase from end to end
- 📱 **Fully responsive** — works on desktop, tablet, and mobile
- 🌗 **Dark mode ready** — light/dark theme support out of the box
- 🧩 **Reusable components** — sidebar, navbar, cards, tables, and dashboard widgets
- 🚀 **Easy to deploy** — one-click deploy to Vercel or Netlify

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm, yarn, pnpm, or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/sachidumaleesha/shadcn-dashboard.git

# Move into the project directory
cd shadcn-dashboard

# Install dependencies
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the dashboard.

You can start editing the UI by modifying files inside `src/app`. The page auto-updates as you save changes.
